    <h1>Performance_Evaluvator</h1>
    
    <div class="section">
        <span class="badge csharp">C#</span>
        <span class="badge windows">Windows</span>
        <span class="badge license">MIT License</span>

    <p>
        <strong>Performance_Evaluvator</strong> is a C# desktop application for evaluating employee performance efficiently. Users can enter performance metrics, visualize results with color-coded bars, and calculate grades based on defined criteria.
    </p>

    <div class="section">
        <h2>Features</h2>
        <ul>
            <li><strong>Employee Identification</strong> – Input Employee ID and Name.</li>
            <li><strong>Performance Metrics</strong> – Evaluate Productivity, Teamwork, and other relevant criteria.</li>
            <li><strong>Color-Coded Evaluation</strong> – Bars change color dynamically based on performance.</li>
            <li><strong>Automatic Grade Calculation</strong> – Generates performance grade according to entered metrics.</li>
            <li><strong>Reset & Recalculate</strong> – Clear bars and recalculate anytime.</li>
            <li><strong>User-Friendly Interface</strong> – Easy-to-use Windows Forms interface.</li>
        </ul>
    </div>
